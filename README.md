# Branz AI 🤖

**One AI. Endless Possibilities.**

Branz AI is a next-generation AI platform that enables users to:

- 💬 Chat with AI
- 🎨 Generate images
- 🎬 Animate images
- 🎤 Voice conversations
- 📄 Analyze documents
- 💻 Generate code
- 🧮 Solve mathematics
- 🌍 Translate languages

## Tech Stack

- Frontend: Next.js + TypeScript
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Mobile: Flutter
- AI: OpenAI APIs
